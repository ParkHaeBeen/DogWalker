# 🦮 소개 
바쁜 현대인들을 위해 강아지 산책 대행 알바 예약 서비스 입니다. 위치기준, 이름으로 검색하여 산책 대행자 리스트를 조회하고,
서비스 가능한 날짜에 예약을 진행합니다. 서비스 진행후  산책 경로를 볼 수 있습니다.
